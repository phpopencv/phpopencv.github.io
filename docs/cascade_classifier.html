级联分类器

目标

在本教程中，您将学习如何：
使用 CV\CascadeClassifier 类来检测视频流中的物体. 特别地, 我们将使用函数:

  • load 来加载一个 .xml 分类器文件. 它既可以是Haar特征也可以是LBP特征的分类器.
  • detectMultiScale 来进行图像检测
